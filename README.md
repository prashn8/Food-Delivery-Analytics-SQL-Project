# Food Delivery Analytics SQL Project

## Project Overview

This project analyzes restaurant performance across Swiggy and Zomato using PostgreSQL.

The goal was to extract business insights related to:
- Revenue Analysis
- Restaurant Performance
- Delivery Efficiency
- Market Share Analysis
- Customer Ratings
- Cuisine Analysis

## Tools Used

- PostgreSQL
- SQL
- GitHub

## Dataset

The project contains data from:
- Restaurants
- Swiggy
- Zomato

## SQL Concepts Used

- Joins
- GROUP BY
- HAVING
- CASE WHEN
- Window Functions
- CTEs
- Date Functions

## Business Questions Solved

1. Which city generates the highest revenue?
2. Which platform dominates market share?
3. Does faster delivery improve ratings?
4. Which cuisines are most profitable?
5. Which restaurants perform well on both platforms?
